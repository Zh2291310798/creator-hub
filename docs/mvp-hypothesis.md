# CreatorHub 最小 MVP 假设

> **版本**: v1.0 | **日期**: 2026-06-02 | **阶段**: 1 — 战略与假设

---

## 一、MVP 核心理念

CreatorHub 是一个**社区驱动的创作者对接平台**。MVP 要验证的核心问题是：

> **"人们是否会因为社区归属感，而愿意在一个平台上完成对接、交易和成长？"**

不是验证"对接需求是否存在"（假设已成立），而是验证"社区+对接"这个组合是否成立。

---

## 二、四大核心假设

### 假设 H1：归属感驱动留存

| 项目 | 内容 |
|------|------|
| **假设** | 等级系统 + 个人主页 + 社区互动，能让用户的 7 日留存率 ≥ 30% |
| **前提** | 纯交易工具留存率通常只有 10-15%；社区归属感能将留存提升到 30%+ |
| **验证方式** | 埋点追踪注册后 7 天活跃率，分层对比"有社区互动用户"vs"无社区互动用户"的留存差异 |
| **可证伪** | 如果 7 日留存 < 15%，说明归属感策略失败 |
| **当前状态** | ⚠️ 功能已开发（等级/主页/社区/好友），但没有埋点数据验证 |

### 假设 H2：跨平台对接降低匹配成本

| 项目 | 内容 |
|------|------|
| **假设** | 聚合展示多平台创作者数据，能让品牌方找到匹配达人的时间从"数天"缩短到"数小时" |
| **前提** | 品牌方当前在多个平台手动筛选达人，效率极低 |
| **验证方式** | 埋点追踪"品牌方从搜索到发起对接意向"的耗时和步骤数 |
| **可证伪** | 如果品牌方来了但从未发起对接 → 说明聚合价值不足以驱动行为 |
| **当前状态** | ⚠️ 对接模块已开发，但没有筛选效率数据，没有品牌方实际使用数据 |

### 假设 H3：本地化匹配是差异化需求

| 项目 | 内容 |
|------|------|
| **假设** | 本地商家愿意为"能按城市/区域筛选本地达人"这个功能持续使用 CreatorHub |
| **前提** | 目前没有平台做好本地商家 × 本地达人的撮合 |
| **验证方式** | 发布本地推广需求 → 追踪本地达人响应率 → 追踪商家复购率 |
| **可证伪** | 如果本地商家发布需求后无达人响应，或响应后无成交 → 需求不存在或体验不够 |
| **当前状态** | ❌ 未开发。现有对接模块无地域筛选功能 |

### 假设 H4：社区是新人的入行入口

| 项目 | 内容 |
|------|------|
| **假设** | 刚入行的新人会通过社区讨论、加前辈好友、参与帖子来学习和找到第一份工作 |
| **前提** | 新人当前缺乏圈内入口，渴望人脉和学习路径 |
| **验证方式** | 追踪"注册时选择'新人/学生'身份"的用户的社区参与度和首次对接/求职转化率 |
| **可证伪** | 如果新人注册后 14 天内没有任何社区互动（发帖/评论/聊天）→ 社区对新人的吸引力不够 |
| **当前状态** | ⚠️ 社区/聊天/好友功能已开发，但缺少新人引导路径 |

## 三、MVP 功能对照

| 假设 | 对应功能 | v1.0.0 状态 | 差距 |
|------|---------|:---:|------|
| H1 归属感 | 等级系统、个人主页、社区、好友 | ✅ | 需埋点验证 |
| H2 跨平台对接 | 对接广场、品牌⇄达人匹配、留言意向 | ✅ | 缺少筛选和搜索 |
| H3 本地匹配 | 地域筛选、本地商家需求发布 | ❌ | **待开发** |
| H4 入行入口 | 新人引导、行业讨论区、学习路径 | ❌ | **待开发** |

---

## 四、MVP 的"不做什么"

为了聚焦验证核心假设，以下功能暂不纳入 MVP 范围：

| 不做 | 原因 |
|------|------|
| ❌ 真实支付/结算系统 | 先验证对接意向，成交在站外完成 |
| ❌ 移动端 App | Web 响应式先行，验证需求后再考虑 PWA/App |
| ❌ AI 匹配推荐 | 先验证人工筛选+社区推荐的效率是否足够 |
| ❌ 多语言 | 中国市场聚焦 |
| ❌ 创作者数据第三方认证 | 先靠社区社交资本（等级/好友/帖子质量）建立信任 |

---

## 五、成功标准

MVP 被判定为"验证通过"的最低门槛：

| 指标 | 门槛 | 对应假设 |
|------|------|---------|
| DAU | ≥ 50 | H1 |
| 7 日留存 | ≥ 25% | H1 |
| 月对接意向发起 | ≥ 30 次 | H2 |
| 月社区帖子 | ≥ 50 帖 | H1, H4 |
| 本地需求发布 | ≥ 10 条/月 | H3 |
