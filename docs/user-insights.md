## 二、详细画像

### P1：电商新人 — "阿杰"

```
姓名: 阿杰（林俊杰）
年龄: 22
状态: 今年刚毕业，普通二本市场营销专业
目标: 想做抖音/小红书电商带货，但没有门路
当前处境: 每天刷招聘 App → 都是"需要有经验" → 投了石沉大海
```

**典型一天**:
- 早上刷 Boss 直聘/智联 → 投电商运营、带货主播助理 → "需要有1-3年经验"
- 下午刷抖音看别人直播带货 → 羡慕但不知道怎么开始
- 晚上在知乎/小红书搜"电商新人怎么入行" → 碎片信息，不成体系
- 偶尔在朋友圈发"求大佬带"，但没人回应

**核心痛点**:
1. 🔴 **没有圈子** — 不认识行业内的人，没人带、没人指点
2. 🔴 **不知道学什么** — 网上教程太多太杂，不知道先学什么、学到什么程度能接单
3. 🟡 **找不到入门级工作** — 招聘都要经验，没人给新人机会
4. 🟡 **不知道行业真实情况** — 带货到底挣不挣钱？全职还是兼职？没人告诉真实答案

**对 CreatorHub 的期望**:
- 能找到行业前辈交流，有人指路
- 能看到真实的行业讨论——薪资、坑、机会
- 能展示"我愿意学"的态度，而不是被经验门槛筛掉
- 有入门级的兼职/项目机会可以积累经验

### P2：本地生活团购达人 — "大刘"

```
姓名: 大刘（刘洋）
年龄: 28
状态: 本地探店博主，抖音 1.2W 粉，主要做本地餐饮/娱乐
变现方式: 团购分佣 + 偶尔商家付费探店
月收入: ¥3K-12K，极不稳定
```

**典型一天**:
- 上午翻大众点评/抖音找新店 → 主动联系商家谈探店合作
- 中午出门拍 1-2 家店 → 拍素材
- 下午剪视频、发布、挂团购链接
- 晚上回复评论、跟商家结算分佣 → 有的商家拖账不结算

**核心痛点**:
1. 🔴 **找不到本地商家资源** — 每次都要主动去挖新店，效率极低
2. 🔴 **商家不信任小博主** — 粉丝不够多，商家爱答不理
3. 🟡 **分佣结算不透明** — 商家说了算，达人是弱势方
4. 🟡 **本地同行互相不认识** — 各做各的，没法共享商家资源

**对 CreatorHub 的期望**:
- 本地商家能主动发布推广需求，而不是达人一个个去求
- 按城市/区域筛选商家——"我只需要成都锦江区的餐饮商家"
- 能在平台上看到达人的真实探店数据——粉丝少但有转化力的也能被看见
- 认识本地同行，互相推荐商家、交流经验

### P3：各平台博主 & 商家 — "小野" 和 "周姐"

```
姓名: 小野（野原摄影）
年龄: 25
平台: 小红书 2.8W + 抖音 6K + B站 3K
内容: 摄影教程 + 器材评测
变现: 品牌商单（相机/手机厂商）+ 课程售卖
月收入: ¥5K-20K


姓名: 周姐（美美服饰）
年龄: 34
身份: 淘宝服装店主 + 小红书发穿搭
需求: 需要达人帮忙推广新款
月推广预算: ¥3K-8K
```

**博主"小野"的痛点**:
1. 🔴 品牌合作私信散落在 3-4 个平台后台，漏掉很多机会
2. 🔴 不知道报价是否合理——同行什么价位完全不知道
3. 🟡 MCN 抽成太高，但不签又接不到大品牌
4. 🟡 希望在圈内交流——设备、技术、报价都可以聊

**商家"周姐"的痛点**:
1. 🔴 想找穿搭博主推广，但不知道怎么筛选——小红书搜出来好多都是刷的
2. 🔴 预算有限（3K-8K），大博主请不起，小博主找不到
3. 🟡 发了几个博主合作，效果参差不齐，不知道怎么判断谁值得投

**共同期望**:
- 博主端：一个聚合面板看所有平台合作机会 + 同行的报价参考
- 商家端：按预算/类目/地区筛选达人 + 参考真实数据做决策

### P4：新媒体从业者 & 品牌方 — "阿 Ken" 和 "张经理"

```
姓名: 阿 Ken
年龄: 26
身份: 做了 2 年短视频编导，最近被裁
目标: 找下一份全职或者先接兼职过渡
技能: 脚本/拍摄/剪辑都能做，也会直播


姓名: 张经理
年龄: 36
身份: 某护肤品品牌市场部负责人
需求: 新品上市需要批量找达人做种草推广
预算: ¥10-30万/月
```

**阿 Ken 的痛点**:
1. 🔴 Boss 直聘上"新媒体编导"岗位太笼统——有的要短视频、有的要公众号、有的要直播，岗位需求不精准
2. 🔴 想找主播类工作（兼职/全职），但没有专门的渠道——BOSS 上主播岗信息混乱
3. 🟡 有作品但没法在一个地方完整展示——投简历只能贴链接，HR 不一定点开看
4. 🟡 想在行业圈子里保持活跃——认识同行、交流行情、不脱节

**张经理的痛点**:
1. 🔴 每次新品上线要定向找达人——按肤质/地域/风格筛选，现有渠道做不到
2. 🔴 达人数据真假难辨——MCN 报的数据水分太大
3. 🟡 管理 20+ 个达人合作，全靠 Excel + 微信——排期、内容审核、结算全手工
4. 🟡 希望达人主动来投——而不是每次都品牌方去挖

**共同期望**:
- 求职者：专注新媒体/直播行业的招聘渠道，能展示作品，能标"可全职/可兼职"
- 品牌方：主动发布需求，达人主动投，双向选择

## 三、用户核心需求总结

根据用户反馈，五条核心需求主线：

| # | 需求主线 | 对应画像 | CreatorHub 现有支持 |
|---|---------|---------|-------------------|
| 1 | **入行学习** — 没入圈的新人想学习提升，需要行业知识和前辈指导 | P1 电商新人 | ❌ 待建设（社区可部分满足） |
| 2 | **品牌↔达人对接** — 品牌需要达人推广，达人需要品牌商单 | P3 博主&商家、P4 品牌方 | ⚠️ 有对接模块但需优化 |
| 3 | **本地商家↔本地达人** — 按地域匹配，本地推广资源对接 | P2 本地达人 | ❌ 待建设（需要地域筛选） |
| 4 | **找工作/招人** — 主播/新媒体从业者找全职兼职，品牌方招人 | P1 电商新人、P4 求职者 | ⚠️ 有招募模块但需细分 |
| 5 | **圈内社交** — 行业交流、交朋友、讨论行情 | 全部画像 | ⚠️ 有社区+聊天，待激活 |

## 四、当前行为 vs 期望行为

| 场景 | 当前行为（无 CreatorHub） | 期望行为（有 CreatorHub） |
|------|--------------------------|--------------------------|
| 新人入行学习 | 知乎/小红书搜碎片信息 → 没人带 | 社区看行业讨论 → 加前辈好友请教 → 找到入门机会 |
| 达人找商单 | 刷各平台私信 → 加微信 → 来回谈 | 开 CreatorHub → 看对接广场 → 一键表达意向 |
| 品牌找达人 | 平台手动搜索 → 逐个联系 → Excel 管 | 发布需求 → 达人主动投 → 筛选 → 平台内跟进 |
| 本地商家推广 | 在抖音/大众点评搜本地博主 → 私信 | 按城市筛选达人 → 发布本地推广需求 → 匹配 |
| 新媒体找工作 | Boss 直聘刷 → 投简历 → 等消息 | 展示作品主页 → 标注"可全职/兼职" → 品牌主动联系 |
| 圈内社交 | 微信群/朋友圈 → 熟人介绍 | 社区发帖讨论 → 聊天 → 加好友 → 互相推荐 |

## 五、关键洞察

1. **"入行"比"对接"更前置** — 大量新人第一需求不是接单，是找到组织、学习路径和人脉入口
2. **"本地"是差异化突破口** — 本地商家×本地达人的地域匹配，现有平台都没做好
3. **"找工作"和"找达人"是同一枚硬币的两面** — 品牌方要人，从业者要工作，招聘模块应该双向驱动
4. **社交是所有需求的地基** — 学习、对接、找工作、本地匹配，最终都落在"先认识人，再办事"
