# CreatorHub 冷启动方案

> **版本**: v1.0 | **日期**: 2026-06-03 | **作者**: P10/CTO
> **问题**: 双边市场没有初始供给 = 鬼城

---

## 一、问题拆解

两边市场冷启动的死锁：

```
品牌方不来 → 因为没有达人
        ↓
    没有达人 → 因为没有品牌需求
        ↓
    没人发帖 → 因为没人看
        ↓
    鬼城 → 访客 30 秒跳出
```

解法不是"同时获取两边用户"，而是**先单边启动供给，用供给拉需求**。

---

## 二、策略：三步冷启动

### Phase 1：伪造供给 → 吸引达人（Week 1-2）

**为什么不先拉品牌？** 品牌方来看到空荡荡没人，不会再回来。达人忍耐力更高——他们愿意逛、愿意发帖。

**种子供给清单**（预置，非 mock 标签）：

| 类型 | 数量 | 来源 |
|------|------|------|
| 品牌需求 | 20-30 条 | 从小红书/抖音手动搬运真实品牌招募帖，标注"搬运自公开渠道" |
| 招募岗位 | 15-20 条 | 从 Boss 直聘/即刻搬运创作类岗位 |
| 本地商家需求 | 10-15 条 | 从大众点评/小红书搬运本地探店需求 |
| 行业讨论帖 | 30-40 条 | 原创撰写（AI 辅助），覆盖报价/避坑/工具/政策 |

**关键设计**：每条搬运内容标注来源链接 + `[搬运]` 标签。不是骗人，是"汇总之地"——这本身就是价值。

**配套功能**：
- 首页加"🔥 最新需求"滚动条（优先展示真实用户发的，种子内容降权）
- 社区首页置顶"📌 新人必读：如何接到第一个商单"

### Phase 2：邀请机制 → 冷启动达人（Week 2-4）

**目标**：获取前 100 个真实创作者

**邀请入口**：
```
个人主页 → "邀请朋友入驻" → 生成专属邀请链接/海报
好友列表 → "导入通讯录" → 发邀请
```

**激励设计**：
- 邀请 1 人注册 → +50 XP
- 被邀请人发第一帖 → 邀请人 +30 XP
- 被邀请人完成第一次对接 → 邀请人 +100 XP + "伯乐"徽章
- 前 100 名入驻创作者 → "创始创作者"徽章（永久展示）

**不做的事情**：
- 不做现金激励（吸引薅羊毛的）
- 不做强制邀请（裂变传销感）

### Phase 3：品牌入驻 → 正循环（Week 4+）

品牌方看到：
1. 达人列表有真实创作者（种子 + 邀请来的）
2. 社区有活跃讨论
3. 对接需求有真实意向留言

→ 品牌方愿意发布第一条真实需求 → 达人收到真实对接 → 正循环启动

## 三、产品功能设计

### F-CS1：邀请系统

```
数据模型：
- invitation_code: { code, inviter_username, created_at, used_by, used_at }
- 邀请链接格式: https://creatorhub.vercel.app/?invite=ABC123

流程：
1. 邀请人点"邀请朋友" → 生成唯一 code
2. 分享链接/海报
3. 被邀请人打开链接 → 注册自动绑定邀请关系
4. 被邀请人发帖/对接 → 邀请人收到通知 + XP
5. 邀请人个人主页展示"已邀请 X 人入驻"
```

### F-CS2：种子内容管理

```
┌─────────────────────────────────────┐
│ 种子内容标识系统                      │
│                                      │
│ 种子帖子: [搬运] 标签 + 来源链接       │
│ 种子需求: [示例] 标签 + "来自公开渠道" │
│                                      │
│ 排序降权规则:                         │
│ - 种子内容默认排在真实内容后面          │
│ - 当本周真实内容 ≥ 10 条时,种子内容折叠 │
│ - 当本周真实内容 ≥ 50 条时,种子内容隐藏 │
└─────────────────────────────────────┘
```

### F-CS3：新人引导强化

```
注册完成 → 三步引导:

Step 1: "选择你关注的平台" (多选：小红书/抖音/B站/快手/视频号/YouTube/TikTok)
  → 首页信息流按平台偏好排序

Step 2: "关注推荐前辈" (基于身份推荐 5 人，一键关注)
  → 新人首页有内容可看

Step 3: "发布你的第一条帖子" (新人报到模板预填)
  → 新人立即有参与感
```

| 邀请转化率 | — | — | 15% | 20% |
| 次日留存 | — | 20% | 25% | 30% |

## 五、不做的事情

- 不在冷启动阶段投广告（先验证自然增长）
- 不做强制手机号注册（保持低门槛）
- 不做虚假用户/机器人（种子内容是搬运的，但用户是真的）
- 不在冷启动阶段做品牌端 BD（先有达人供给再 BD 品牌）
