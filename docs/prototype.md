# CreatorHub 原型图 & 信息架构

> **版本**: v1.0 | **日期**: 2026-06-02 | **阶段**: 1 — 战略与假设

---

## 一、信息架构（Sitemap）

```
CreatorHub
│
├─ 🔐 登录/注册（Auth Overlay）
│   ├─ 登录表单
│   ├─ 注册表单（用户名 + 密码 + 身份选择）
│   └─ 身份选择：品牌方 / 创作者 / 招聘方 / 自由职业
│
├─ 🎪 首页（Home）
│   ├─ 浮动 emoji 背景
│   ├─ 平台芯片墙（小红书/抖音/B站/快手...）
│   ├─ 实时统计滚动（在线用户/对接成功/社区帖子）
│   ├─ 身份快速切换卡
│   └─ 快捷入口：聊天 / 招募 / 对接 / 社区
│
├─ 📡 动态广场（Square / Feed）
│   └─ 实时信息流（招募/对接/帖子动态）
│
├─ 💬 聊天（Chat）
│   ├─ 联系人列表
│   ├─ 私聊窗口（模拟回复）
│   └─ 🌍 世界频道
│
├─ 💼 招募（Recruit）
│   ├─ 招募信息列表（按平台筛选）
│   ├─ 发布招募表单
│   └─ 招募详情弹窗
│
├─ 🤝 对接（Match）
│   ├─ 品牌方视图 → 找达人
│   ├─ 达人视图 → 找品牌
│   ├─ 筛选（平台/类目）
│   ├─ 意向留言弹窗
│   └─ 发布对接需求
│
├─ 📝 社区（Community）
│   ├─ 帖子列表（按分类筛选）
│   ├─ 发帖编辑器
│   ├─ 评论/删帖
│   └─ 全屏帖子详情
│
├─ 👥 好友（Friends）
│   ├─ 好友列表
│   ├─ 好友请求
│   ├─ 发现新朋友
│   └─ 好友资料弹窗
│
├─ 👤 个人主页（Profile）
│   ├─ 资料展示（头像/等级/平台标签）
│   ├─ 我的帖子
│   ├─ 好友列表
│   └─ 编辑资料入口
│
├─ ⚙️ 设置（Settings）
│   ├─ 编辑资料（昵称/简介/平台/标签）
│   ├─ 头像选择器（16渐变 + 20表情）
│   ├─ 通知设置
│   └─ 隐私设置
│
├─ 🆕 【待开发】本地匹配（Local Match）  ← H3
│   ├─ 城市/区域筛选
│   ├─ 本地商家需求发布
│   ├─ 本地达人展示
│   └─ 地域标签
│
└─ 🆕 【待开发】新手引导（Onboarding）  ← H4
    ├─ 入行身份选择（学生/新人/转行）
    ├─ 行业话题推荐
    ├─ 推荐关注的前辈列表
    └─ "入门第一课"内容区
```

## 二、现有页面 → 假设映射

| 页面 | 覆盖假设 | 状态 | 说明 |
|------|:---:|:---:|------|
| 首页 | H1 | ✅ | emoji+统计+平台墙 = 品牌感 + 社区温度 |
| 动态广场 | H1 | ✅ | 实时信息流，让用户感知"有人在这里" |
| 聊天 | H1, H4 | ✅ | 私聊+世界频道，社交基础 |
| 招募 | H2, H4 | ✅ | 招聘/求职双向，但缺新媒体/主播细分 |
| 对接 | H2 | ⚠️ | 品牌⇄达人匹配已实现，缺地域筛选 |
| 社区 | H1, H4 | ⚠️ | 发帖评论已实现，缺新人引导和行业分区 |
| 好友 | H1 | ✅ | 好友关系+发现，社交资本 |
| 个人主页+设置 | H1 | ✅ | 等级+平台标签+头像，归属感载体 |
| **本地匹配** | **H3** | ❌ | **待建——核心差异化功能** |
| **新手引导** | **H4** | ❌ | **待建——入行入口** |

## 四、用户流程

### 4.1 品牌方找达人（H2）
```
首页 → 对接 → 筛选（平台/类目/🆕城市）→ 浏览达人卡片
                                             ├→ 💬 留言意向
                                             └→ 👤 查看达人主页 → 加好友 → 聊天
```

### 4.2 本地商家找推广（H3）
```
首页 → 📍本地对接 → 选择城市/区域 → 筛选品类
                                      ├→ 浏览本地达人 → 邀约
                                      └→ 发布需求 → 等达人接单
```

### 4.3 新人入行（H4）
```
注册（选学生/新人身份）→ 🆕新手首页 → 读入门文章
                                     ├→ 关注前辈
                                     ├→ 在 #新人报道 发第一帖
                                     └→ 看招募/对接，找到第一个机会
```

### 4.4 达人接商单（H2）
```
首页 → 对接（达人视图）→ 浏览品牌需求 → 💬 留言意向
                                        └→ 聊天沟通 → 站外成交
```

| # | 页面/流程 | 假设 | 验证方式 |
