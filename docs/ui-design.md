> **版本**: v1.0 | **日期**: 2026-06-02 | **阶段**: 3 — 技术设计

## 一、设计系统：Builder Jam 手账风

### 设计理念
- **有温度的纸质感**：便签、手写字体、圆角不规则边框，像一本创意手账
- **低心理门槛**：不像 BOSS 直聘那样正式，更像创作者的"秘密基地"
- **色彩克制 + 强调色点缀**：大面积暖白/米色 + 黑墨色文字 + 平台品牌色点缀

## 三、组件库

### 3.1 按钮

```
.btn                    ← 基础按钮（白底黑边 + 硬阴影）
.btn.big               ← 大按钮（min-height: 46px）
.btn.green             ← 绿色变体（确认/成功操作）
.btn.red               ← 红色变体（删除/危险操作）
.btn.secondary         ← 次要按钮（灰色背景）

状态：
  default:  白底 + 4px 黑边 + 硬阴影
  hover:    translateY(-2px) + 阴影加深
  active:   translateY(0) + 阴影减弱
  disabled: 灰色文字 + 灰边 + 无阴影

示例：
  <button class="btn">普通按钮</button>
  <button class="btn big green">确认</button>
  <button class="btn secondary">取消</button>
```

### 3.2 卡片

```
.card                   ← 基础卡片（白底 + 波浪边 + 硬阴影）
  hover: 轻微上浮 2px + 阴影增强

变体：
  - 达人卡片：头像 48px + 等级徽章 + 平台标签 + 操作按钮
  - 帖子卡片：作者头像 + 分类标签 + 正文预览 + 评论数
  - 需求卡片：标题 + 品类标签 + 城市标签 + 预算 + 操作按钮
  - 商家卡片：商家名 + 品类图标 + 城市 + 需求 + 预算

标签样式：
  .tag-platform → 平台品牌色圆角标签
  .tag-category → 灰色圆角标签
  .tag-city     → 🆕 带 📍 图标的城市标签
  .tag-level    → 等级徽章（渐变色）
```

### 3.3 输入框

```
.input                  ← 基础输入（白底 + 波浪边 + 底部聚焦线）
  default: 灰边
  focus:   蓝色底部线 + 轻微放大
  error:   红色边框
  disabled: 灰色背景

.input-area            ← 多行文本域（min-height: 80px）
```

### 3.4 弹窗

```
.overlay               ← 全屏半透明背景（背景模糊）
.dialog / .drawer      ← 白色面板 + 波浪边 + 大阴影
  - .dialog: 居中，宽 90vw max 480px
  - .drawer: 底部弹出，高 60vh

弹窗头部：标题（手写体）+ [×] 关闭按钮
弹窗内容：可滚动
弹窗底部：操作按钮区
```

### 3.5 Toast

```
.toast                 ← 底部居中浮动条
  白底 + 黑边 + 硬阴影 + 圆角
  出现：fadeIn + translateY(-20px)
  消失：fadeOut + translateY(0)
  自动消失：2-3 秒
```

### 3.6 徽章/标签

```
.level-badge           ← 等级徽章（渐变色底 + 白色文字）
  Lv.1-3: 灰色系
  Lv.4-6: 蓝色系
  Lv.7-9: 紫色系
  Lv.10:  金色渐变

.platform-chip         ← 平台芯片（平台品牌色底 + 白色文字）
  hover: 放大 105%

🆕 .city-tag           ← 城市标签（灰色底 + 📍 前缀）
🆕 .work-status-tag    ← 工作状态标签
  fulltime:  绿色 "🟢 可全职"
  parttime:  蓝色 "🔵 可兼职"
  project:   橙色 "🟠 项目制"
  not_looking: 灰色 "⚫ 暂不求职"
```

### 3.7 头像

```
.avatar                ← 圆形头像 48px（可缩放）
  16 套渐变色 + 20 个 emoji 表情
  无头像时：用户名哈希 → 默认渐变色 + 首字 emoji
```

## 四、布局系统

### 整体布局

```
┌─────────────────────────────────────┐
│  顶部导航栏（可选，当前无）          │
├─────────────────────────────────────┤
│                                     │
│  主内容区（可滚动）                  │
│  - 各 section-panel 通过             │
│    display:none/block 切换           │
│  - 面板内独立滚动                    │
│                                     │
├─────────────────────────────────────┤
│  底部导航栏（固定 8 个入口）          │
└─────────────────────────────────────┘
```

### 响应式断点

| 断点 | 布局 |
|------|------|
| ≥ 768px | 完整桌面布局，卡片 2-3 列 |
| 375-767px | 单列布局，卡片全宽，底部导航缩小 |
| < 375px | 基本可浏览（非重点优化） |
