## 一、全局交互模式

### 1.1 导航

```
┌──────底部导航栏（固定）────────────────────────┐
│  ≡首页 │ 📡广场 │ 💬聊天 │ 💼招募 │ 🤝对接 │ 📝社区 │ 👥好友 │
└────────────────────────────────────────────────┘
```

| 交互 | 行为 |
|------|------|
| 点击底部导航 | 即时切换面板，无 loading |
| 页面内快捷入口 | 首页卡片/按钮 → 跳转目标面板 |
| 用户头像/名称 | 点击 → 查看该用户主页 |
| 返回 | 浏览器无 SPA 返回栈（单页应用），通过面包屑或关闭弹窗 |

### 1.2 反馈

| 类型 | 组件 | 持续 | 触发场景 |
|------|------|------|---------|
| 成功提示 | Toast 底部居中 | 2 秒 | 登录成功/发帖成功/操作完成 |
| 错误提示 | Toast 底部居中（红色） | 3 秒 | 操作失败/验证不通过 |
| 升级庆祝 | 纸屑动画 + Toast | 3 秒 | 等级提升 |
| XP 获得 | 飘字 "+50 XP" | 1.5 秒 | 每次获得经验 |
| 确认操作 | `confirm()` 弹窗 | 手动关闭 | 删帖/删除好友 |

### 1.3 弹窗/抽屉

| 类型 | 实现 | 关闭方式 | 使用场景 |
|------|------|---------|---------|
| 全屏详情 | 固定定位覆盖层 | 点击 × 或背景 | 帖子详情（`openPostDetail`） |
| 底部抽屉 | 固定定位从下弹出 | 点击 × 或背景 | 招募详情（`openRecruitDrawer`） |
| 居中弹窗 | 固定定位居中 | 点击 × 或背景 | 意向留言（`openMatchMsg`） |
| 底部面板 | 固定定位从下弹出 | 点击 × 或背景 | 好友资料（`showFriendProfile`）、头像选择器 |

## 二、关键页面交互

### 2.1 首页

```
进入首页
  ├── emoji 自动漂浮动画（CSS animation）
  ├── 统计数字滚动递增（setInterval + 数到目标值停止）
  ├── 平台芯片悬停放大（CSS :hover scale）
  └── 身份切换卡片：点击切换 → Toast 提示 "身份已切换：XXX" → 跳转对接页
```

| 微交互 | 实现 |
|--------|------|
| emoji 漂浮 | `@keyframes float` + 随机起点/延迟 |
| 数字滚动 | `setInterval` 递增到目标值 |
| 平台芯片 hover | `transform: scale(1.05)` + `box-shadow` 增强 |

### 2.2 对接页

```
品牌找达人视图                          达人找品牌视图
┌─ 筛选栏 ──────────────────────┐      ┌─ 筛选栏 ──────────────────────┐
│ 平台[▾] 类目[▾] 🆕城市[▾]     │      │ 平台[▾] 类目[▾] 🆕城市[▾]     │
│ 🆕 粉丝量[▾] 🆕 报价区间[▾]    │      │ 🆕 预算区间[▾]               │
└───────────────────────────────┘      └───────────────────────────────┘
  ↓ 即时筛选（客户端过滤，无需请求）

┌─ 达人卡片 ──────────────────────────────────────────┐
│ [头像 48px]  小鹿  Lv.5  @小红书|美妆   🆕 📍杭州    │
│ 粉丝 5.2K · 报价 ¥1,000-2,000                        │
│                                              [💬 留言意向] │
│                                              [👤 查看主页] │
└──────────────────────────────────────────────────────┘
```

| 微交互 | 行为 |
|--------|------|
| 筛选切换 | 卡片列表即时过滤，过滤结果为空显示"没有找到匹配的达人/品牌" |
| "留言意向"按钮 | 弹出居中弹窗 → 输入留言 → 发送 → Toast "留言已发送" → 对方聊天列表新增消息 |
| "查看主页"按钮 | 切换至 profile 面板并定位到该用户 |
| 发布需求 | 表单展开（toggle）→ 填写 → 发布 → 表单收起 + 列表刷新 |

### 2.3 🆕 本地匹配页

```
进入本地匹配
  ├── 默认选择用户所在城市（从 myProfile.city 读取）
  ├── 如未设置城市 → 弹出城市选择器
  └── 加载该城市的商家需求和达人排行

城市选择器交互：
  点击城市标签 → 下拉面板（省份 → 城市 → 区域三级）
  → 选择后即时刷新列表

商家需求卡片：
  [🍔 餐饮] 火锅店「蜀大侠」· 锦江区
  需求：探店视频 × 3条  ·  预算 ¥500-800/条
  [🤝 我要接]  [💬 咨询]

  点击"我要接" → 弹出意向留言 → 自动填入"我对 XX 商家的探店需求感兴趣"
```

### 2.4 🆕 新手引导

```
注册完成 → 检测 isOnboardingComplete()
  ├── 未完成 → 弹出新手引导覆盖层
  │     Step 1: 👋 欢迎 + 根据身份展示不同欢迎语
  │     Step 2: 📖 推荐入门文章（3-5 篇）
  │     Step 3: 👥 推荐关注前辈（5-8 人）
  │     Step 4: 💡 热门话题 + 引导发"新人报到"帖
  │     Step 5: 🎉 完成 → 发放 +100 XP → 跳转首页
  │
  └── 已完成 → 直接进入首页

引导覆盖层交互：
  - 全屏半透明背景 + 白色卡片
  - "下一步"按钮 + "跳过"链接
  - 进度条（Step 1/5 → 5/5）
  - 最后一步"开始探索"按钮 → 关闭覆盖层
```

### 2.5 社区

```
帖子列表
├── 分类 Tab：[全部] [经验分享] [🆕新人报到] [🆕行业讨论]
└── 帖子卡片
    ├── 点击 → 全屏详情（openPostDetail）
    ├── 全屏内：评论列表 + 评论输入框
    └── 作者本人可见 [🗑 删除] 按钮

行业讨论子标签（筛选时自动过滤）：
  [#报价] [#避坑] [#工具] [#政策]
```

## 三、状态与过渡

### 3.1 加载状态

| 场景 | 处理 |
|------|------|
| 页面切换 | 无加载（客户端即时渲染） |
| 首次打开 | Auth Overlay → 登录 → 渐隐过渡 → 首页 |
| 数据为空 | 显示空状态文案 + 引导行动 |

### 3.2 空状态

| 页面 | 空状态文案 | 引导行动 |
|------|-----------|---------|
| 聊天 | "还没有聊天，去对接广场认识新朋友吧" | [去对接] |
| 好友 | "还没有好友，去发现新朋友吧" | [发现朋友] |
| 社区（全部） | "还没有帖子，成为第一个发帖的人" | [写帖子] |
| 招募 | "暂无招募信息" | [发布招募] |
| 对接 | "暂无匹配结果，试试调整筛选条件" | [清除筛选] |
| 🆕 本地匹配 | "该城市暂无本地需求" | [发布需求] 或 [切换城市] |

### 3.3 错误状态

| 错误 | 处理 |
|------|------|
| localStorage 满 | Toast "存储空间不足，请清理浏览器数据" |
| 数据解析失败 | 静默回退到默认值 + console.warn |
| 操作冲突 | Toast 提示具体原因 |

## 四、动效规范

| 动效 | 时长 | 缓动 | 说明 |
|------|:---:|------|------|
| 面板切换 | 0ms | — | 即时切换（无动画） |
| Toast 进出 | 300ms | ease | fadeIn + slideUp |
| 弹窗进出 | 200ms | ease-out | scale(0.9→1) + fadeIn |
| 纸屑 | 2s | ease-out | particles 从中心向外飘散 |
| XP 飘字 | 1.5s | ease-out | 向上浮动 + 淡出 |
| 数字滚动 | 1s | ease-out | 数字递增 |
| 按钮 hover | 150ms | ease | translateY(-2px) + shadow 增强 |
| 卡片 hover | 200ms | ease | 轻微上浮 |
